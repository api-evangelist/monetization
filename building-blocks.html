These are the common building blocks I have pullled from my research so far.

Free
Educational Access

Educational access is about making resources available to students, and the academic insttutions they attend. Academic pricing allows students to put resources to work, gain a familiarity with services, without having to pay regular fees.
Free

A popular way to provide access to an API, is by offering a free tier so that anyone can sign up, start using an API and understand the value an API delivers. This allows consumers to kick the tires, see if it will meet their needs before spending money.
Free Trial

Rather than just free for an unknown amount of time, so API providers are providing the ability to test drive an API for a period of time, then you have to pay. Usually this is associated with some sort of SaaS business model.
Freemium

Providing a relief for free users, allowing them to upgrade to a paid account, usuallky part of a tiered approach to pricing. Freemium starts as free, allowing users to kick the tires, but once they begin to depend on they can begin paying for the service.
Not For Profit

It is not just Sillicon Valley startups doing APIs, non-profit organizations, and government agencies are doing APIs, and the monetization strategy is not all about making money. 

Consumer Pays
Request Pricing

Services that require a user pays, but does not have pricing listed directly on the site, and users must request a quote for pricing on single or all API access. 
Buy Credits

You haveh to buy credits, charge your user accounts with funds from PayPal, bank account, or credit card before you can exceed basic API rate limits.
Volume Pricing

Providing additional layers of pricing based upon volume purchases, allowing users to better understand their API usage and potentially buy in bulk based upon their past habits.
Pay as You Go

Allowing API consumer to pay as they go, accumulating a monthly bill, or possibly prepaying on their account. Either way they are able to pay for what they us of API resources, as they use it.
Tiered

Providing multiple tiers of paid access, such as bronze, gold or platinum. Each tier has its own set or services and allowances for access to API resources. With pricing, stepping up in cost for each tier
Unit Based

When pricing is based upon individually define units of value made available via an API. This could be per API call, storage on disk, message, or any other pre-defined unit of measurement.
Enterprise Pricing

Enterprise is a broad term to describe pricing that is part of a larger sales process, and a common way for large companies to price products and services based upon a customers need, and information extracted through sales cycles.

Marketing Vehicle

APIs can provide an excellent marketing vehicle for a company and its online presence. Through sensible branding strategies, developers can become 3rd party marketing agents, working on behalf of a core company and its b
Traffic Generation

APIs can also be used to drive traffic to an existing website or applicaiton. By desigining an API to use hyperlinks directed at central websites or apps, and encouraging consumers to build their own websites and apps that are integrated with the API, the opportunity for driving traffic to other sites is very desireable to many companies who are providing APIs.
Brand Awareness

As a new tool in an overall marketing and branding strategy, an API can provide a type type of brand exposure via 3rd party websites and applications. Extending the reach of a brand, using 3rd party API consumers as the engine.
Content Acquisition

Content acquisition is when a platform uses APIs to enable the adding, updating, and deleting of content for the purpose of employing users to generate content for a core platform.
SaaS

Software as a Service (SaaS) has become a common approach to selling software online to consumers and businesses.  Often times an API will compliment the core software and its offering, providing value to SaaS users. API access is often included as part of core SaaS platform, but also can be delivered as an upsell for premium SaaS users.
