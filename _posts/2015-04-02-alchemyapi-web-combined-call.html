AlchemyAPI Web Combined Call is available in pro and enterprise packages and is used to process publicly-accessible Internet content, including hosted webpages and blogs.This API performs multiple analysis operations on a given web page by downloading the requested URL, removing unecessary content such as navigation links and advertisements, and extracting text from the HTML document structure.Requests are in JSON and responses are in JSON, RDF, or XML.Authentication is through unique and private API Key.AlchemyAPI is a data mining platform.
